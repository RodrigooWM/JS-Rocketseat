Prototype

Usa-se o elemento, termo, etc + Ponto (.) para indicar um prototype.

Type conversion, casting x coersion

type conversion ou type casting ocorre quando alteramos manualmente o tipo de um elemento no código.

Ex.: Number('200') → Estamos transformando uma string em um number.

type coersion ocorre quando o próprio js altera o tipo de um elemento no código.

Ex.: console.log('200' + 100) → O número 100 virará uma string ao ser apresentado na tela.

Quando usamos?

Podemos alterar os tipos por diversos motivos. O principal deles é quando precisamos identificar o comprimento (length)de uma sequência numérica. Para isso, a sequência numérica deve ser alterada para string antes.

Casas decimais

Usamos o método "toFixed(número de casas)" para determinar o número de casas decimais que um número terá.

Ex.: 

        let string = '200.6564'
        console.log(Number(string).toFixed(3))

        Retorna: 200.656

Se quisermos trocar o ponto pela vírgula, usamos o comando .replace(".",",") Ex.: console.log(Number(string).toFixed(3).replace(".",","))

Texto em caixa alta ou baixa

var.toUpperCase() / var.toLowerCase()

Separando strings

            let  frase = 'Quero sair do Brasil.'
            let myArray = frase.split(" ") (Dessa forma a separação será feita de acordo com cada espaço em branco que houver.)
            console.log(myArray)
Retorno: [ 'Quero', 'sair', 'do', 'Brasil.' ]

Podemos separar também usando um elemento da frase: 
            let  frase = 'Quero sair do Brasil.'
            let myArray = frase.split("do")
            console.log(myArray)
Retorno: [ 'Quero sair ', ' Brasil.' ]

Mais um exemplo: 
            let  frase = 'Quero sair do Brasil.'
            let myArray = frase.split("a")
            console.log(myArray)
Retorno: [ 'Quero s', 'ir do Br', 'sil.' ]

Podemos também separar uma frase acrescentando sinais, da seguinte forma: 

        let frase = "Eu quero sair do Brasil!"
        let myArray = frase.split(" ")
        let sepfrase = myArray.join("_")
        console.log(sepfrase)

Retorno: Eu_quero_sair_do_Brasil!
Obs.: O underline (_) também pode ser trocado por outros elementos.

Encontrando elementos em um texto:

Usamos o comando var.includes("termo que queremos encontrar").

Ex.:

                let frase = "Eu quero sair do Brasil!"
                console.log(frase.includes("sair"))

Retorno: true
Obs.: O elemento difere com maiúsculas e minúsculas.

Manipulando arrays

Os comandos abaixo representam a inserção de elementos, remoção, identificação da posição, etc. Revise e pratique.

    techs.push("SQL")

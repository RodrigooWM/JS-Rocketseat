Variáveis

As palavras reservadas para variáveis são "let, const e var". A diferença entre elas é que var e let podem mudar, mas const não. Uma vez que atribuirmos uma variável usando a palavra reservada "const", ela não pode ser alterada.

Lembre-se que os comandos além de serem escritos em maiúscula após cada espaço (getElementById), também podem ser escritos com o snake case (get_element_by_id). Faça o teste.

Acessar dados de uma variável

        const person= {
            name:'jon',
            age:30,
            weight:97.30,
            isAdmin:true
        }
        
        console.log(person.name)

Usando pontos podemos definir um caminho até a informação que desejamos.

Acessar dados de uma variável quando array

        const animals= [
    'Lion',
    'Monkey',
    'Cat'
]

console.log(animals[2])

Devemos definir a posição do termo dentro do array. Lembrando que o array começa a contagem em 0 e não em 1.

Piorando o mesmo caso:

        const animals= [
    'Lion',
    'Monkey',
    {
        name: 'Lucca',
        age: 3 + ' aninhos'
    }
]

console.log(animals[2].name + ' idade ' + animals[2].age)

Retorno:
Lucca idade 3 aninhos

Funções JS

Sabemos que o formato de uma função em JS é
function nomedela(){
}

Para iniciarmos uma função, basta habilitarmos a mesma na linha do JS - nomedela().

Ex.:

        function notes() {
        }
        notes()
Ao chegar na linha "notes()", a função será executada. Podemos repetir esse comando o número de vezes que quisermos que a função seja repetida.

Passando parâmetros/argumentos para a função

            const sum = function(n1,n2){
                res = n1 + n2
            }
            sum(2,3)
            console.log(res)

Código estranho abaixo:

Arrow function com callback

        function feijao (name) {
            name()
        }
        feijao(
            () => {
                console.log('Estou em chamada')
            }
        )

Function constuctor

function Person(name){
        this.name = name
    }
    const rodri = new Person('Rodrigo')
    const joao = new Person ('João')
    console.log(rodri)
    console.log(joao)

Retorno:
Person { name: 'Rodrigo' }
Person { name: 'João' }

Alteração na função

            function Person(name){
                this.name = name
                this.walk = function(){
                    return "andando"
                }
            }
            const rodri = new Person('Rodrigo')
            const joao = new Person ('João')
            console.log(rodri.walk())
            console.log(joao.walk())
